Create `proxy/test/test-install.sh` — a shell script that:
1. Creates a temp copy of `proxy/com.ailinks.claude-proxy.plist` (which will exist after Plan 02 creates it).
2. Runs the substitution logic from `install.sh` (sed replacement of __HOME__ and __PROXY_DIR__).
3. Asserts that the output plist does NOT contain `__HOME__` or `__PROXY_DIR__` literal strings.
4. Asserts that the output plist DOES contain the actual `$HOME` value.
5. Prints PASS or FAIL and exits with code 0 or 1.

```bash
#!/bin/bash
set -e
PROXY_DIR="$(cd "$(dirname "$0")/.." && pwd)"
PLIST_SRC="$PROXY_DIR/com.ailinks.claude-proxy.plist"

if [ ! -f "$PLIST_SRC" ]; then
  echo "SKIP: plist template not yet created (run after Plan 02)"
  exit 0
fi

TMPFILE=$(mktemp /tmp/test-plist.XXXXXX)
sed -e "s|__HOME__|$HOME|g" -e "s|__PROXY_DIR__|$PROXY_DIR|g" "$PLIST_SRC" > "$TMPFILE"

if grep -q "__HOME__\|__PROXY_DIR__" "$TMPFILE"; then
  echo "FAIL: plist still contains placeholders"
  rm "$TMPFILE"
  exit 1
fi

if ! grep -q "$HOME" "$TMPFILE"; then
  echo "FAIL: plist does not contain actual HOME path"
  rm "$TMPFILE"
  exit 1
fi

rm "$TMPFILE"
echo "PASS: plist substitution correct"
```
